# نموذج تسجيل الأطباء — MedicalLinks (العربية)

## أ. التقديم السريع

### المعلومات الشخصية
- الاسم الكامل
- البريد الإلكتروني
- رقم الجوال
- رقم واتساب
- الجنسية
- دولة الإقامة الحالية
- المدينة الحالية

### الملخص المهني
- المسمى المهني (طبيب عام / طبيب مقيم / أخصائي / استشاري / أخرى)
- التخصص الطبي
- التخصص الدقيق (اختياري)
- إجمالي سنوات الخبرة
- سنوات الخبرة بعد الحصول على التخصص إن وجدت
- جهة العمل الحالية
- دولة الممارسة الحالية

### اللغة والانتقال
- مستوى اللغة الإنجليزية (مبتدئ / جيد / جيد جداً / ممتاز)
- مستوى اللغة العربية (لا يوجد / مبتدئ / جيد / ممتاز)
- الدول الخليجية المفضلة (يمكن اختيار أكثر من دولة)
  - السعودية
  - الإمارات
  - قطر
  - الكويت
  - عُمان
  - البحرين
- أقرب تاريخ ممكن للبدء
- هل أنت مستعد للانتقال؟ (نعم / لا)

### المستندات والموافقة
- رفع السيرة الذاتية
- اختياري: رفع نسخة من الترخيص المهني
- مربع موافقة:
  - أؤكد أن المعلومات المقدمة صحيحة بحسب علمي وأوافق على تواصل MedicalLinks معي بخصوص الفرص الطبية المناسبة.

## ب. الملف الكامل بعد الفرز الأولي

### 1. المعلومات الشخصية
- الاسم الكامل
- تاريخ الميلاد
- الجنس (اختياري)
- الجنسية
- الحالة الاجتماعية (اختياري)
- عدد الأطفال (اختياري)
- دولة جواز السفر
- عنوان السكن الحالي

### 2. بيانات التواصل
- رقم الجوال
- رقم واتساب
- البريد الإلكتروني
- لغة التواصل المفضلة (العربية / الإنجليزية)

### 3. المؤهلات المهنية
- الدرجة الطبية الأساسية
- الجامعة / المؤسسة
- دولة التخرج
- سنة التخرج
- هل أتممت سنة الامتياز؟ (نعم / لا)
- التخصص
- التخصص الدقيق
- الدرجة / المستوى الحالي
- تفاصيل البورد أو شهادة التخصص
- دورات أو شهادات إضافية

### 4. الخبرة العملية
- الحالة الوظيفية الحالية
- جهة العمل الحالية
- المسمى الوظيفي الحالي
- دولة الممارسة الحالية
- إجمالي سنوات الخبرة السريرية
- سنوات الخبرة بعد التخصص
- مدة الإشعار
- أقرب موعد للانضمام

### 5. اللغات
- مستوى اللغة الإنجليزية
- مستوى اللغة العربية
- لغات أخرى

### 6. التراخيص والاعتمادات
- التراخيص الطبية الحالية
- الدول التي تحمل بها ترخيصاً
- أرقام التسجيل
- هل شهادة حسن السيرة المهنية متوفرة؟ (نعم / لا / جاري الإجراء)
- حالة DataFlow (لم يبدأ / قيد الإجراء / مكتمل / لا ينطبق)
- حالة اختبار Prometric أو ما يعادله
- حالة التصنيف المهني في أي جهة صحية خليجية
- هل سبق لك العمل في الخليج؟ (نعم / لا)
- إذا كانت الإجابة نعم، أين ومتى؟

### 7. تفضيلات سوق العمل
- الدول الخليجية المفضلة
- المدن المفضلة (اختياري)
- نوع الجهة الصحية المفضل
  - مستشفى
  - عيادة
  - مؤسسة أكاديمية
  - ممارسة خاصة
  - أخرى
- نوع العقد المفضل
- هل أنت منفتح على أكثر من دولة خليجية؟ (نعم / لا)

### 8. الراتب والمزايا
- الراتب الحالي الشهري أو السنوي (اختياري)
- نطاق الراتب المتوقع
- أهم المزايا بالنسبة لك
  - السكن
  - التأمين الصحي
  - تذاكر السفر
  - دعم تعليم الأبناء
  - دعم الانتقال
  - الإجازة السنوية
  - التطوير المهني

### 9. الجاهزية للانتقال
- هل تحتاج إلى كفالة / تأشيرة عمل؟ (نعم / لا)
- هل ستنتقل بمفردك أم مع العائلة؟ (اختياري)
- هل توجد أي قيود رئيسية على الانتقال؟ (اختياري)
- الإطار الزمني المفضل للانتقال

### 10. المراجع المهنية
- المرجع الأول: الاسم، المسمى، الجهة، البريد الإلكتروني، الهاتف
- المرجع الثاني: الاسم، المسمى، الجهة، البريد الإلكتروني، الهاتف

### 11. قائمة المستندات
- السيرة الذاتية
- نسخة جواز السفر
- شهادة التخرج
- شهادة الامتياز
- شهادة التخصص / البورد
- نسخة الترخيص الطبي
- شهادة حسن السيرة المهنية
- شهادات الخبرة
- تقرير DataFlow إن وجد
- نتيجة الاختبار إن وجدت

### 12. الإقرار النهائي
- أؤكد أن جميع المعلومات المقدمة صحيحة وكاملة بحسب علمي.
- أفوض MedicalLinks بمراجعة مؤهلاتي والتواصل مع مراجعي لأغراض التوظيف.

- التاريخ
